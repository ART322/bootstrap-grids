Grid System

The Bootstrap grid system is applied by adding classes to your HTML.

It needs a basic HTML structure using div tags, with classes applied to those tags

It is a 12 column grid system
(all the column widths have to equal to 12 in a row)

It's Responsive â€“ it converts to a stacked view on mobile devices

Columns are inside Rows
Rows are inside a Container


You should know...

The column classes on Bootstrap have a specific notation that determines what happens to the grid at different screen sizes.


xs display grid at phone size and above
sm display grid at tablet size and above
md display grid at desktop size and above
lg display grid at larger desktop size and above

For the purposes of this project you are going to primarily use sm:

<div class="col-sm-4">
<img src="images/gradient.jpg" class="img-responsive">
</div>

We want a grid to appear at any size larger than a phone
